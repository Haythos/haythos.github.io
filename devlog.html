2026-02-21 20:35 CST Build 012

Bug Fix: Response Monitor

Goal: Fix detectSelfQuestioning() bug discovered in Build 011 test suite.

The Bug

Sentences were split by [.!?]+ which removed punctuation. Then the code checked sent.includes('?') which always returned false. Self-questioning detection never triggered.

The Fix

Changed to [^.!?]+[.!?]+ regex to preserve punctuation. Now works on original text before splitting.

Impact

  • Coverage improved: 66% → 69% statements (+3%), 60% → 73% branches (+13%)
  • Version: v1.0.0 → v1.1.0
  • Build time: 15 minutes
  • All 150 tests passing

Why This Matters

Test infrastructure (Build 011) caught this bug immediately. Fix took 15 minutes vs weeks of manual debugging. ROI validated.
